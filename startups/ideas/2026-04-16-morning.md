### A. Signal Quality Scoring Algorithm

```python
def calculate_signal_quality(interaction):
    score = 0.0
    
    # Explicit signals (highest weight)
    if interaction.explicit_rating:
        score += 0.4 * normalize(interaction.explicit_rating)
    
    # Behavioral signals
    if interaction.user_edited:
        edit_quality = analyze_edit(
            original=interaction.output,
            edited=interaction.edited_version
        )
        score += 0.3 * edit_quality
    
    # Outcome signals
    if interaction.downstream_outcome:
        outcome_value = measure_outcome(interaction.downstream_outcome)
        score += 0.2 * outcome_value
    
    # Contextual signals
    score += 0.1 * context_relevance(interaction)
    
    return score
```
