The Threat

Prompt Injection Is the New SQL Injection

AI tools read untrusted content and execute commands. Attackers are already exploiting this.

📋

GitHub Issue Injection

Malicious prompts hidden in issue titles and descriptions trigger AI tools to execute arbitrary code when developers view them.

📦

Dependency Poisoning

Attackers embed prompts in package docstrings. When AI reads the codebase, it follows malicious instructions.

🔄

PR Review Attacks

Hidden prompts in pull request descriptions can manipulate AI code reviewers into approving malicious changes.

📁

Repository Cloning

Cloning repos with embedded prompts in READMEs or config files triggers immediate AI execution.
