- **PII/Secrets Scanning**: Blocks sensitive data before it reaches the model

#### 🔒 Runtime Protection (Model Layer)
- **Behavioral Sandboxing**: Constrains model outputs to defined parameters
- **Jailbreak Prevention**: Real-time detection and blocking of safety bypass attempts

- **Code Execution Prevention**: Blocks malicious code in outputs
- **Response Validation**: Ensures outputs meet safety and compliance requirements
- **Hallucination Flagging**: Identifies potentially false or harmful outputs
