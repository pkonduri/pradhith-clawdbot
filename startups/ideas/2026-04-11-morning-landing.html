Windows → Linux Migration

The playbook powering France's €2.3B government transformation

1

Discovery 4-6 weeks

Deploy Discovery AI agents across Windows estate. Map all applications, dependencies, and integrations. Identify Linux equivalents and gaps. Create user workflow documentation.

2

Pilot 8-12 weeks

Select pilot group (typically IT + early adopters). Transform AI converts applications and configurations. Deploy Linux environment with parallel access. Validate AI confirms functionality.

3

Wave Migration Ongoing

Department-by-department migration waves with zero-downtime cutover. Continuous validation and support. Rollback capability maintained throughout.

4

Decommission 4-8 weeks

Legacy system monitoring for stragglers. Data archival and compliance. License termination. Migration completion certification.
