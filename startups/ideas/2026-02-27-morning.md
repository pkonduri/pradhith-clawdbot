### API Integration

Simple drop-in for any AI pipeline:

```python
from factchain import FactChain

fc = FactChain(api_key="...")

# Verify AI output
result = fc.verify(
    content="The FDA approved Drug X in 2024...",
    context="medical_advice",
    confidence_threshold=0.85
)

print(result.fact_score)  # 94.2
print(result.sources)     # [PubMed, FDA.gov, ...]
print(result.provenance)  # Cryptographic proof chain
```
